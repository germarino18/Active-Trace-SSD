Sidebar navigation row. Active = violet text on a raised surface with a right violet accent bar.

```jsx
<NavItem icon="dashboard" label="Dashboard" active />
<NavItem icon="menu_book" label="Mis Materias" badge="6" />
<NavItem icon="person" label="Mi Perfil" />
```

`icon` is a Material Symbols name. `badge` shows a violet count chip. Hover raises the background one surface tier.
